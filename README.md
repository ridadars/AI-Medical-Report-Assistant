# 🩺 AI Medical Report Assistant

An AI-powered web application that helps users understand their medical reports.

## Features

- Upload PDF medical reports
- OCR text extraction
- AI explanation of reports
- Highlight abnormal values
- Health history dashboard
- Secure authentication

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- FastAPI

### AI
- Google Gemini API
- EasyOCR

### Database
- PostgreSQL
